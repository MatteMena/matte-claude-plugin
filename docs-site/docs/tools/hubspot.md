# HubSpot Tools

All tools for managing HubSpot CRM contacts, companies, deals, and tickets.

## Account Tools

| Tool | Operation | Description |
|------|-----------|-------------|
| `list-hubspot-accounts` | Read | List connected HubSpot portals |

## Contact Tools

| Tool | Operation | Description |
|------|-----------|-------------|
| `list-hubspot-contacts` | Read | List all contacts |
| `get-hubspot-contact` | Read | Get a specific contact by ID |
| `search-hubspot-contacts` | Read | Search contacts by property |
| `create-hubspot-contact` | Write | Create a new contact |
| `update-hubspot-contact` | Write | Update contact properties |
| `delete-hubspot-contact` | Delete | Delete a contact |
