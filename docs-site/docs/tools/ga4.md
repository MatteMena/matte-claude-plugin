# Google Analytics (GA4) Tools

All tools for querying Google Analytics 4 properties. All GA4 tools are read-only.

## Account Tools

| Tool | Operation | Description |
|------|-----------|-------------|
| `list-google-analytics-connections` | Read | List all GA4 connections |
| `get-ga4-properties` | Read | List all GA4 properties |
| `get-ga4-audiences` | Read | List GA4 audiences / segments |

## Reporting Tools

| Tool | Operation | Description |
|------|-----------|-------------|
| `get-ga4-traffic-overview` | Read | Sessions, users, page views, bounce rate |
| `get-ga4-channel-grouping` | Read | Performance by marketing channel |
| `get-ga4-top-pages` | Read | Most visited pages with engagement metrics |
| `get-ga4-conversions` | Read | Conversion events and rates |
| `get-ga4-geo-report` | Read | Performance by country and region |
| `get-ga4-device-report` | Read | Desktop, mobile, tablet breakdown |
| `get-ga4-acquisition-report` | Read | New vs returning users by source/medium |
| `get-ga4-landing-pages` | Read | Landing page performance |
| `get-ga4-events-report` | Read | Custom and automatic event tracking |
| `get-ga4-retention-report` | Read | User retention and cohort analysis |
| `get-ga4-funnel-report` | Read | Step-by-step funnel analysis |
| `get-ga4-attribution-report` | Read | Multi-touch attribution models |
| `get-ga4-ecommerce-report` | Read | Revenue, transactions, product performance |
| `get-ga4-realtime-report` | Read | Active users and events right now |
| `get-ga4-report` | Read | Custom report with any dimensions and metrics |
