# TikTok Tools

All tools for managing TikTok for Business ad accounts, campaigns, ad groups, ads, creatives, audiences, and pixels.

## Account Tools

| Tool | Operation | Description |
|------|-----------|-------------|
| `list-tiktok-connections` | Read | List all TikTok advertiser connections |
| `get-tiktok-advertisers` | Read | List all advertiser accounts |
| `get-tiktok-advertiser` | Read | Get details for a specific advertiser |
| `get-tiktok-account-summary` | Read | Get performance summary for an advertiser |

## Campaign Tools

| Tool | Operation | Description |
|------|-----------|-------------|
| `get-tiktok-campaigns` | Read | List campaigns for an advertiser |
| `create-tiktok-campaign` | Write | Create a new campaign |
| `update-tiktok-campaign` | Write | Update campaign settings |
| `update-tiktok-campaign-status` | Write | Enable, pause, or archive a campaign |
| `delete-tiktok-campaigns` | Delete | Delete one or more campaigns |

## Ad Group Tools

| Tool | Operation | Description |
|------|-----------|-------------|
| `get-tiktok-ad-groups` | Read | List ad groups for a campaign |
| `create-tiktok-ad-group` | Write | Create a new ad group |
| `update-tiktok-ad-group` | Write | Update ad group settings |
| `delete-tiktok-ad-groups` | Delete | Delete one or more ad groups |

## Ad Tools

| Tool | Operation | Description |
|------|-----------|-------------|
| `get-tiktok-ads` | Read | List ads for an ad group |
| `get-tiktok-spark-ads` | Read | List Spark Ad posts |
| `create-tiktok-ad` | Write | Create a new ad |
| `update-tiktok-ad` | Write | Update ad settings |
| `delete-tiktok-ads` | Delete | Delete one or more ads |

## Insights Tools

| Tool | Operation | Description |
|------|-----------|-------------|
| `get-tiktok-insights` | Read | Get performance metrics with date ranges and breakdowns |
