# Snapchat Tools

All tools for managing Snapchat Ads campaigns, ad squads, creatives, audiences, and performance stats.

## Account & Campaign Tools

| Tool | Operation | Description |
|------|-----------|-------------|
| `get-snapchat-organizations` | Read | List Snapchat Business organizations |
| `get-snapchat-ad-accounts` | Read | List ad accounts |
| `get-snapchat-campaigns` | Read | List campaigns |
| `create-snapchat-campaign` | Write | Create a new campaign |
| `update-snapchat-campaign` | Write | Update campaign settings |

## Ad Squad & Ad Tools

| Tool | Operation | Description |
|------|-----------|-------------|
| `get-snapchat-ad-squads` | Read | List ad squads |
| `create-snapchat-ad-squad` | Write | Create a new ad squad |
| `get-snapchat-ads` | Read | List ads |
| `create-snapchat-ad` | Write | Create a new ad |

## Audience Tools

| Tool | Operation | Description |
|------|-----------|-------------|
| `get-snapchat-audiences` | Read | List custom audiences |
| `create-snapchat-audience` | Write | Create a custom audience |

## Insights Tools

| Tool | Operation | Description |
|------|-----------|-------------|
| `get-snapchat-stats` | Read | Get campaign, ad squad, or ad performance stats |
