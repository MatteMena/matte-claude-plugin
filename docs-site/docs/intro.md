# What is MatteMena?

**MatteMena** connects [Claude](https://claude.ai) to your advertising platforms so you can research, create, and optimise campaigns using natural language — no dashboard-switching required.

## How it works

1. **Sign up** at [nerve.mattemena.com](https://nerve.mattemena.com) and connect your ad accounts.
2. **Install** the MatteMena plugin in Claude Code, or add the connector in Claude.ai.
3. **Chat** — ask Claude to pull reports, create campaigns, adjust budgets, or analyse performance.

## Supported Platforms

| Platform | What you can do |
|----------|----------------|
| **Meta (Facebook/Instagram)** | Campaigns, ad sets, ads, creatives, audiences, pixels, insights |
| **TikTok Ads** | Campaigns, ad groups, ads, audiences, pixels, insights, Spark Ads |
| **Google Ads** | Campaigns, ad groups, keywords, bidding, budgets, PMax, insights |
| **Google Analytics (GA4)** | Traffic, channels, conversions, funnels, e-commerce, attribution |
| **Snapchat Ads** | Campaigns, ad squads, creatives, audiences, stats |
| **Google Sheets** | Read and write spreadsheet data |
| **HubSpot CRM** | Contacts, companies, deals, tickets |

## Tool Discovery Mode

MatteMena uses **Tool Discovery Mode** by default. Instead of exposing 185+ individual tools, Claude is given 4 discovery tools that let it automatically find the right tool for any task and run multiple tools in parallel.

This means:
- Faster responses (parallel execution)
- Smarter tool selection (semantic search)
- No need to manually pick tools

[Learn more about Tool Discovery Mode →](/docs/tools/discovery)

## Authentication

MatteMena uses **OAuth 2.0 with PKCE** (the same standard as "Login with Google"). You authenticate once via your browser — no API keys to copy or store.

[Authentication guide →](/docs/authentication)
