# Google Sheets

Connect Claude to your Google Sheets to read and write spreadsheet data directly from your conversation.

## Connecting Google Sheets

1. Go to [nerve.mattemena.com](https://nerve.mattemena.com) → **Connected Sources**
2. Click **Connect Google Sheets**
3. Authorise access to your Google Drive and Sheets

## What you can do

### Reading Data
- List all your connected spreadsheets
- Get a full spreadsheet or specific worksheet
- Read rows, cells, or ranges
- Find rows matching specific criteria

### Writing Data
- Add single or multiple rows
- Update existing rows and individual cells
- Create new spreadsheets
- Delete or clear rows

## Example prompts

```
"Read all rows from my 'Campaign Tracker' spreadsheet"
"Add a new row to my 'Weekly Report' sheet with today's Meta spend"
"Find all rows in my sheet where the Status column is 'Pending'"
"Create a new spreadsheet called 'Q3 Ad Performance'"
"Update cell B2 in 'Budget Tracker' to 5000"
```

## Use cases

- Export Claude's ad performance analysis to a Google Sheet
- Read your media plan sheet to brief Claude on upcoming campaigns
- Sync campaign data between platforms via Claude
- Build reporting dashboards by having Claude write data to Sheets
