# Google Analytics (GA4)

Connect Claude to your Google Analytics 4 properties to run reports on traffic, channels, conversions, funnels, e-commerce, and attribution — all in natural language.

## Connecting GA4

1. Go to [nerve.mattemena.com](https://nerve.mattemena.com) → **Connected Sources**
2. Click **Connect Google Analytics**
3. Authorise access to your Google account
4. Select the GA4 properties you want Claude to access

## What you can do

| Report Type | Description |
|-------------|-------------|
| **Traffic Overview** | Sessions, users, page views, bounce rate |
| **Channel Grouping** | Performance by channel (Organic, Paid, Direct, Social) |
| **Top Pages** | Most visited pages with engagement metrics |
| **Conversions** | Conversion events and rates |
| **Geo Report** | Performance by country/region |
| **Device Report** | Desktop, mobile, tablet breakdown |
| **Acquisition Report** | New vs returning users by source |
| **Landing Pages** | Landing page performance |
| **Events Report** | Custom event tracking |
| **Retention Report** | User retention and cohort data |
| **Funnel Report** | Step-by-step funnel analysis |
| **Attribution Report** | Multi-touch attribution |
| **E-commerce Report** | Revenue, transactions, product performance |
| **Realtime Report** | Active users right now |
| **Custom Report** | Any dimensions and metrics you specify |

## Example prompts
